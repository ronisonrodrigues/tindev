## Inicialização

Backend:

```sh
yarn dev
```

Frontend:

```sh
yarn start
```

Mobile (Android):

Primeira execução para instalar o app:
```sh
yarn react-native run-android
```
Após a primeira instalação:
```sh
yarn start
```

Mobile (iOS):

Primeira execução para instalar o app:
```sh
yarn react-native run-ios
```
Após a primeira instalação:
```sh
yarn start
```
